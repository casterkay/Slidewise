Welcome to this tutorial on React Hooks. Today we'll cover the most important hooks you need to know, starting with useState and useEffect, and then building our own custom hooks.

What are Hooks?

  • Functions that let you use state and React features in functional components
  • Before hooks, class components were needed for state management
  • Only call hooks at the top level and from React functions
Hooks are functions that let you use state and other React features in functional components. Before hooks, you needed class components for state management. The key rules are: only call hooks at the top level, and only call hooks from React functions.

useState Deep Dive


const [count, setCount] = useState(0);

// Multiple state variables
const [name, setName] = useState('');
const [items, setItems] = useState([]);
  • Returns array with current state and setter
  • Use multiple useState for different state
Let's look at useState. You import it from React, then call it with an initial value. It returns an array with the current state and a setter function.

useEffect Patterns

  • Perform side effects after render
  • Dependency array controls when it runs
  • Empty array = run once on mount
  • Return cleanup function to avoid memory leaks
useEffect lets you perform side effects in your components. It runs after every render by default. You can control when it runs by passing a dependency array.

Custom Hooks


function useWindowSize() {
  const [size, setSize] = useState(window.innerWidth);
  useEffect(() => {
    const handler = () => setSize(window.innerWidth);
    window.addEventListener('resize', handler);
    return () => window.removeEventListener('resize', handler);
  }, []);
  return size;
}
  • Extract reusable logic with use prefix
  • Can call other hooks and compose nicely
Custom hooks let you extract reusable logic. Name them with the use prefix.

Key Takeaways

  1. useState for component state management
  2. useEffect for side effects with dependency control
  3. Custom hooks for reusable logic (use prefix)
  4. Only call hooks at top level, from React functions
That's it for today's tutorial on React Hooks. Remember: useState for state, useEffect for side effects, and custom hooks for reusable logic.
